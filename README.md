# First-part-of-final-project-10gr
## Пользовательские сценарии
### Регистрация и вход
Главная страница  
      ↓  
Пользователь нажимает "Регистрация"  
      ↓  
Заполняет форму (имя, email, пароль)  
      ↓  
Нажимает "Создать аккаунт"  
      ↓  
Аккаунт создается  
      ↓  
Пользователь попадает в ленту  
### Вход в аккаунт  
Главная страница  
      ↓  
Нажимает "Войти"  
      ↓  
Вводит email и пароль  
      ↓  
Нажимает "Войти"  
      ↓  
Открывается лента  
### Просмотр сообщений  
Лента  
   ↓  
Пользователь видит список постов  
   ↓  
Нажимает на пост  
   ↓  
Открывается страница поста  
   ↓  
Можно читать комментарии  
### Отправка сообщения (поста)  
Лента  
   ↓  
Нажимает "Создать пост"  
   ↓  
Пишет текст  
   ↓  
Нажимает "Опубликовать"  
   ↓  
Пост появляется в ленте  
### Создание мероприятия  
Меню → Календарь  
      ↓  
Нажимает "Создать событие"  
      ↓  
Вводит название, дату, описание  
      ↓  
Нажимает "Создать"  
      ↓  
Событие появляется в календаре

### Конечные точки на сервере и контракты данных   
#### Авторизация  
Запрос  
```json
{
 "username": "stepa",
 "email": "stepa@mail.com",
 "password": "123456"
}
```
Ответ  
```json
{
 "message": "User created",
 "user_id": 1
}
```  
#### Получение постов  
```json
{
  "id": 1,
  "author": "stepa",
  "content": "Привет лицеистам!",
  "created_at": "2026-03-08"
}
```  
#### Создание поста  
Запрос
```json
{
 "author_id": 1,
 "content": "Сегодня будет встреча клуба программирования"
}
```
Ответ
```json
{
 "message": "Post created",
 "post_id": 5
}
```
#### Получить комментарии поста
```json
{
  "id": 1,
  "author": "ivan",
  "content": "Круто!",
  "created_at": "2026-03-08"
}
 ```
#### Добавить комментарий
Запрос
```json
{
 "post_id": 1,
 "author_id": 2,
 "content": "Отличный пост!"
}
```
Ответ
```json
{
 "message": "Comment created",
 "comment_id": 10
}
```
#### Получить список групп
```json
{
  "id": 1,
  "name": "Клуб программирования",
  "description": "Для тех, кто любит программировать"
}
```
#### Создать группу
Запрос
```json
{
 "name": "Клуб робототехники",
 "description": "Создаем роботов",
 "creator_id": 1
}
```
Ответ
```json
{
 "message": "Group created",
 "group_id": 4
}
```
#### Получить мероприятия
```json
[
 {
  "id": 1,
  "title": "Хакатон",
  "description": "Соревнование программистов",
  "date": "2026-04-10"
 },
 {
  "id": 2,
  "title": "Шахматный турнир",
  "description": "Турнир среди учеников",
  "date": "2026-04-15"
 }
]
```
